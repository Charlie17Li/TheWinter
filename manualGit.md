#### 让一个项目具有Git功能
> git init

#### 从github克隆一个仓库
> git clone username@host:/path/to/repository

#### 添加和提交
> git add file  
> git commit -m "commitment"

#### 推送
> git push origin master 可以改成任何master以外的分支
